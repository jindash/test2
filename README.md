# 运维查询工具（Ops Query Tool）

基于 **React + Bootstrap + Axios + Express** 的前后端示例项目，实现：

- 角色登录（管理员/普通用户）
- 基于角色的动态菜单与多标签页工作区
- 系统配置（数据库配置、业务菜单配置、脚本配置）
- 业务单据查询（模块化查询、脚本参数动态表单、多结果集展示）

## 启动

```bash
npm install
npm run dev
```

- 前端：http://localhost:5173
- 后端 API：http://localhost:3001/api

## 默认账号

- 管理员：`admin / admin123`
- 普通用户：`user / user123`
